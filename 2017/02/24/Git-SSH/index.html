Hexo+Git 搭建Blog时需要注意的地方

  • Git
  • Hexo

对之前的博文进行补充：
（1）在安装完Git之后，需要对SSH进行配置

  • 检查SSH key

  • 备份已有的key（如果有的话）

  • 生成SSH key

  • 将SSH key添加到Github
    登录到GitHub页面，Account Settings->SSH Public Keys->Add another key将生成的key(id_rsa.pub文件）内容copy到输入框中，save。

  • 测试连接

  • 设置个人信息

（2）修改hexo根目录下_config.yml文件（xxx为你的github账户名称）
